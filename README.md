# Grocery-Store

In this Python project, we will build a **grocery store management application**. It will be a **3-tier application**:

1. **Front-end**: UI written in **HTML/CSS/JavaScript/Bootstrap**.
2. **Back-end**: **Python** and **Flask**.
3. **Database**: **MySQL**.
